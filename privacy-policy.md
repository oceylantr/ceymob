## Overview

Bonibon Icon Pack is built using the open-source CandyBar dashboard framework ([github.com/zixpo/candybar](https://github.com/zixpo/candybar)).  
**We do not collect, store, or share any personal user data.**

---

## Permissions Used

The app requests the following Android permissions. None of them are used to collect personal data.

| Permission | Purpose |
|---|---|
| `INTERNET` | Downloading cloud wallpapers; checking for icon updates |
| `ACCESS_NETWORK_STATE` | Checking network availability before loading remote content |
| `ACCESS_WIFI_STATE` | Optimizing data usage on Wi-Fi connections |
| `READ_EXTERNAL_STORAGE` | Reading wallpapers stored on device (Android 12 and below) |
| `SET_WALLPAPER` | Applying wallpapers selected by the user |
| `FOREGROUND_SERVICE` | Running wallpaper download tasks in the background |

---

## Icon Requests

When you submit an icon request, the app generates an email pre-filled with basic device information (launcher name, Android version). This email is sent **directly from your email client** to our support address. We do not intercept or store this data automatically.
